# Robinctl 

This utility is used to control the [Robin](https://github.com/pulcy/robin) load-balancer at runtime.

## Usage 

### List frontends 

```
robinctl ls 
``` 

### Show frontend 

```
robinctl get id <...> 
``` 

### Remove frontends 

```
robinctl rm id <...>
``` 

### Add frontend 

```
robinctl add <options> 
```
